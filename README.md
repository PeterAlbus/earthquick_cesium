# vue_cesium

## Project setup
```
npm install
```
此外需要下载cesium放置于public/cesium文件夹中(链接: [百度网盘，提取码：gsks](https://pan.baidu.com/s/1qFh139dx3T8EoYHMJ5BnLw?pwd=gsks))

tileset文件也没有上传到github，需要自行放置于public/tileset文件夹中（暂时没有用到）
